- 超級月亮
- 電影鏡頭
- 多機聯錄

- 專業模式照片風格

※由於未在各機型做過測試，若有功能出現問題(尤其是機型浮水印)還請不吝嗇回報，感謝您~

### 注意事項
- Magisk直接刷入即可使用
- 僅支援MIUI12.5開發版以上版本的機型使用
- 自v4.3.003591.0之後的版本MIUI13的穩定版可以正常使用，在此之前的版本必定會閃退
- 第三方移植的MIUI ROM無法使用(移植到停更或其它品牌的手機)
- xiaomi.eu ROM可以使用，其餘官改自測
