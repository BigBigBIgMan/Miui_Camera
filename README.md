# [Magisk模組]Miui相機功能補全-Miui_Camera
### 模組介紹
- 萌拍3.0
- 超級月亮
- 電影鏡頭
- 手勢拍照
- 錄影音源變焦
- 錄影HDR模式
- 錄影超級防手震(Pro)
- 多機聯錄
- 專業模式中照片風格
- 前置拍照2X變焦
- 無聲街拍(拍照+錄影)
- MI10&MI10S機型開啟慢動作480FPS
- MI10S機型開啟多攝同開(三個畫面)
- MI10系列修改為新版相機浮水印
- MI12&CIVI系列加入前攝補光介面

### 注意事項
- Magisk直接刷入即可使用
- 僅支援MIUI12.5開發版以上版本的機型使用
- 穩定版無法使用(因為底層架構不同)
- 第三方移植的MIUI ROM無法使用
