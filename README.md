# [Magisk模組]Miui相機功能補全-Miui_Camera
### 模組介紹
- 手勢拍照
- 錄影音源變焦
- 專業模式中照片風格
- 前置拍照2X變焦
- 無聲街拍(連拍+錄影)
- 部分機型修改為新版相機浮水印(※)
- MI12&CIVI系列加入前攝補光介面

※新版浮水印機型:小米10/小米10 Pro/小米10 Ultra/小米10S/小米11/小米11 Pro/小米11 Ultra/小米12系列/紅米K30/紅米K30 Pro/紅米K40
(cmi/umi/cas/thyme/venus/mars/star/picasso/lmi/alioth)

### 僅部分機型可使用功能(避免出現錯誤)
- 萌拍3.0
- 超級月亮
- 電影鏡頭
- 錄影HDR模式
- 錄影超級防手震(Pro)
- 多機聯錄

### 注意事項
- Magisk直接刷入即可使用
- 僅支援MIUI12.5開發版以上版本的機型使用
- 穩定版無法使用(因為底層架構不同)
- 第三方移植的MIUI ROM無法使用
